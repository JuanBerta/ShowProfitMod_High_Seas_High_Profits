# ShowProfitMod: High Seas, High Profits!
A lightweight UI enhancement for High Seas, High Profits! that adds real-time profit calculations to the trade window. This mod helps you make smarter trading decisions by visualizing your margins at a glance.

## 🚀 Features
Sell Profit Tracking: Automatically calculates the difference between a city's buy price and your current Average Cost per unit.

Buy "Deal" Logic: Compares a city's selling price against the Global Base Price to highlight if you're getting a bargain or paying a premium.

Dynamic UI Integration: Injects custom text elements directly into the game's trade rows, keeping the information perfectly aligned with your cargo.

## 🛠 Installation
This mod requires MelonLoader to run.

Download the latest ProfitTextMod.dll from the Releases page.

Navigate to your game folder (usually SteamLibrary\steamapps\common\High Seas, High Profits!).

Place the .dll file inside the /Mods folder.

Launch the game!
